# /design — UI/UX Design and Prototyping

---

## description: Activates Sparks Persona for UI/UX tasks

> **Skill**: [designing-ui](../skills/ui-ux-design/designing-ui/SKILL.md)
> **Goal**: Create premium, "wow" factor interfaces.

## When to Use

- "Design a landing page"
- "Create a component for [feature]"
- "Make this look better"

## Workflow

### Step 1: Activate Sparks (Designer Mode)

Load **`skills/ui-ux-design/designing-ui/SKILL.md`**:

- **Role**: Sparks (The Designer).
- **Style**: Modern, Premium, Dynamic.
- **Constraints**: Mobile-first, Accessible.

### Step 2: Vibe Check

**Ask**:

- What is the aesthetic? (e.g., Cyberpunk, Minimalist, Corporate)
- Any reference sites?
- Color preference?

### Step 3: Scaffold & Implement

1.  **Scaffold**: Create `index.css` vars or layout structure.
2.  **Component**: Build strict, reusable components.
3.  **Polish**: Add hover states, transitions, micro-interactions.

### Step 4: Verify

- **Run**: `/ui-audit` (Vibe Check).
- **Check**: Responsive on mobile? Dark mode support?

---

## Output

- React/Vue Components
- CSS/Tailwind Code
- Design Token definitions
